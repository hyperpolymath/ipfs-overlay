== Scope and Standard

Evaluates the `developer-ecosystem/ipfs-overlay` repository against
*Component Readiness Grades v2.0 (STRICT)*, 2026-03-30 revision.
ipfs-overlay is self-declared *C* in its flat `crg-grade` key and has

estate dogfooding signal, the HYP-S005 overclaim-canary rule, and the
v2.0 "honest D > dishonest B" clause.

evidence supports E or D". It matches the *flat* `crg-grade = "C"`
key under `[metadata]` in STATE.a2ml and cross-checks it against
per-suite test counts, completion-percentage consistency, per-directory
annotation coverage, and dogfooding evidence.

1. *Completion-percentage contradiction*: STATE says 50 %, README

   Core functionality / Basic documentation / CI/CD pipeline".

No HYP-S005 suppression comment is present and none is warranted.
The *test-count* axis of HYP-S005 does *not* fire (suite counts are
honest — this is the one rule ipfs-overlay passes cleanly).
